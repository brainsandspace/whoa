# whoa-cli [![Build Status](https://travis-ci.org/davvidbaker/whoa-cli.svg?branch=master)](https://travis-ci.org/davvidbaker/whoa-cli)

> a CLI for whoa


## Install

```
$ npm install --global whoa-cli
```


## Usage

```js
$ whoa-cli --help

  Usage
    whoa-cli [input]

  Options
    --name  Lorem ipsum [Default: false]

  Examples
    $ whoa-cli
    I love Ink
    $ whoa-cli --name=ponies
    I love ponies
```
